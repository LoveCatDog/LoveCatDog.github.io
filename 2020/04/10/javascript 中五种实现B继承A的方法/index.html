javascript 中五种实现B继承A的方法

1.继承第一种方式：对象冒充

javascript

2.继承第二种方式：call()方法

第一个参数的值赋值给类中出现的this，第二个参数开始依次赋值给类所接受的参数

javascript

3.继承第三种方式：apply()方法

第一个参数为赋值给类中出现的this，第二个参数为数组类型

javascript

4.继承第四种方式：原型链方式

即子类通过prototype 将所有在父类中通过prototype追加的属性和方法都追加到Child4，从而实现继承

javascript
